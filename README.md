# openclaw-skills

My OpenClaw skills collection.

## Skills

### feishu-task

Manage Feishu (Lark) tasks via the v2 API. Supports creating, updating, deleting tasks, managing members, reminders, tasklists, subtasks, and more.

**Setup**: Replace `YOUR_APP_ID` and `YOUR_APP_SECRET` in `scripts/api.py` with your Feishu app credentials.

### session-cost

Analyze OpenClaw session logs for token usage, costs, and performance metrics grouped by agent and model.

```bash
node scripts/session-cost.js --details
```
